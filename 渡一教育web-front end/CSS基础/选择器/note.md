## 选择器的组合
1. 并且 如：p.red两者组合，中间不需要空格
2. 后代元素 - 空格
如.red li{ }表示选中了class为red的后代元素为li
3. 子元素  > 如：.red > bcd
4. 相邻兄弟元素 .元素+ 比如：.red+li是选择.red属性所在元素的下一个li
5. 后面出现的所有的兄弟元素 -～ 比如：.red~li

# 选择器的并列
多个选择器，用逗号分隔
如：.red~li,p{} 当两个元素内的格式一样的时候可以使用
